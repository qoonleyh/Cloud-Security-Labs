Azure Firewall Policy via Portal

Overview

This lab demonstrates how to create and configure Azure Firewall policies through the Azure Portal, including network rules, application rules, and threat intelligence settings to control and inspect traffic flowing through the firewall.

Services Used


Azure Firewall
Azure Firewall Policy
Azure Virtual Network (VNet)
Azure Virtual Machines
Azure Monitor


Steps Performed


Deployed an Azure Firewall in a hub VNet
Created an Azure Firewall Policy via the Azure Portal
Configured network rule collections to allow and deny specific traffic
Configured application rule collections for FQDN-based filtering
Enabled threat intelligence-based filtering
Tested traffic flows and verified firewall rule enforcement
