Microsoft Sentinel Content Hub

Overview

This lab demonstrates how to use the Microsoft Sentinel Content Hub to discover, deploy, and manage out-of-the-box security content including analytics rules, workbooks, playbooks, and data connectors for common security scenarios.

Services Used


Microsoft Sentinel
Sentinel Content Hub
Log Analytics Workspace
Azure Logic Apps
Azure Monitor


Steps Performed


Navigated to the Microsoft Sentinel Content Hub
Searched for and installed a content solution package
Deployed analytics rules from the installed solution
Configured workbooks included in the content package
Reviewed and activated playbooks from the solution
Verified deployed content appeared in the respective Sentinel sections
