Microsoft Sentinel

Overview

This lab provides hands-on experience with Microsoft Sentinel, Azure's cloud-native SIEM and SOAR solution. It covers workspace setup, data connector configuration, analytics rules, and incident investigation.

Services Used


Microsoft Sentinel
Log Analytics Workspace
Azure Monitor
KQL (Kusto Query Language)
Azure Active Directory


Steps Performed


Created a Log Analytics Workspace and enabled Microsoft Sentinel
Connected data connectors to ingest security logs
Created analytics rules to detect suspicious activity
Generated test alerts and reviewed Sentinel incidents
Wrote KQL queries to investigate incidents in the Log Analytics workspace
Explored Sentinel workbooks for security visualisation
