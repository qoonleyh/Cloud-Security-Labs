Azure Front Door (ARM & Bicep)

Overview

This lab demonstrates how to deploy Azure Front Door using both ARM Templates and Bicep to provide global load balancing, SSL offloading, and Web Application Firewall (WAF) capabilities for web applications.

Services Used


Azure Front Door
Azure Web Application Firewall (WAF)
ARM Templates
Bicep (Infrastructure as Code)
Azure Virtual Machines / App Services


Steps Performed


Deployed Azure Front Door using an ARM Template
Configured Front Door backend pools and routing rules
Enabled WAF policy on the Front Door profile
Redeployed using a Bicep template for comparison
Tested global load balancing and WAF rule enforcement
Verified SSL termination and health probe configuration
