Azure SQL Database Security

Overview

This lab demonstrates how to secure an Azure SQL Database using Transparent Data Encryption (TDE), auditing, Advanced Threat Protection, and firewall rules to protect sensitive data at rest and in transit.

Services Used


Azure SQL Database
Azure SQL Server
Transparent Data Encryption (TDE)
Azure Monitor / Auditing
Azure Key Vault


Steps Performed


Created an Azure SQL Server and SQL Database
Enabled Transparent Data Encryption (TDE) on the database
Configured SQL Server auditing to write logs to a Storage Account
Enabled Microsoft Defender for SQL (Advanced Threat Protection)
Configured SQL Server firewall rules to restrict access by IP
Reviewed audit logs and security recommendations
