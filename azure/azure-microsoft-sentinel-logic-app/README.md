Microsoft Sentinel Logic App (SOAR)

Overview

This lab demonstrates how to configure automated incident response in Microsoft Sentinel using Azure Logic Apps as SOAR playbooks, enabling automated actions triggered by security alerts and incidents.

Services Used


Microsoft Sentinel
Azure Logic Apps
Log Analytics Workspace
Azure Monitor
Azure Active Directory


Steps Performed


Created a Microsoft Sentinel workspace with active analytics rules
Created an Azure Logic App playbook for automated incident response
Configured the Logic App with Sentinel trigger and response actions
Attached the playbook to a Sentinel analytics rule as an automated response
Triggered a test incident and verified the Logic App executed correctly
Reviewed the Logic App run history and Sentinel incident timeline
