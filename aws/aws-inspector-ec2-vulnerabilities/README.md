AWS Inspector — EC2 Vulnerability Assessment

Overview

This lab demonstrates how to use Amazon Inspector to perform automated vulnerability assessments on EC2 instances, identifying software vulnerabilities and unintended network exposure.

Services Used


Amazon Inspector
Amazon EC2
AWS Systems Manager (SSM)
Amazon Web Services (IAM)


Steps Performed


Launched an Amazon Linux 2 EC2 instance with SSM Agent enabled
Enabled Amazon Inspector and configured EC2 scanning
Ran an Inspector vulnerability assessment on the EC2 instance
Reviewed Inspector findings including CVE references and severity ratings
Explored recommended remediation steps for identified vulnerabilities
