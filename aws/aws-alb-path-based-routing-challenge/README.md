Path Based Routing — AWS ELB Challenge

Overview

A challenge lab requiring independent implementation of path-based routing on an AWS Application Load Balancer without step-by-step guidance, demonstrating proficiency in ALB listener rule configuration.

Services Used


Amazon EC2
Elastic Load Balancing (ALB)
Amazon VPC
Security Groups


Steps Performed


Launched two Amazon Linux 2 EC2 instances independently
Created three ELB target groups with healthy EC2 targets
Configured an Application Load Balancer with a multi-rule HTTP listener
Implemented path-based routing rules for admin and user paths
Validated ELB DNS accessibility and correct routing behaviour


Outcome
