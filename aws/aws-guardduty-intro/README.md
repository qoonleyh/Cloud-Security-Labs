Introduction to Amazon GuardDuty

Overview

This lab provides an introduction to Amazon GuardDuty, AWS's intelligent threat detection service. It covers enabling GuardDuty, reviewing findings, and understanding how GuardDuty analyses CloudTrail, VPC Flow Logs, and DNS logs to detect threats.

Services Used


Amazon GuardDuty
AWS CloudTrail
Amazon VPC Flow Logs
Amazon S3


Steps Performed


Enabled Amazon GuardDuty in the AWS Management Console
Reviewed the GuardDuty dashboard and findings panel
Generated sample GuardDuty findings for review
Explored finding types including reconnaissance, instance compromise, and account compromise
Reviewed finding severity levels and recommended remediation actions
