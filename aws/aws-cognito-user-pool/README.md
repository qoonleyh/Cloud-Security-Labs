Creating a User Pool in AWS Cognito

Overview

This lab covers creating and configuring an Amazon Cognito User Pool with custom password policies, MFA settings, account recovery options, and email configuration to provide secure user authentication for applications.

Services Used


Amazon Cognito
Amazon Web Services (IAM)


Steps Performed


Navigated to Amazon Cognito in the AWS Management Console
Created a new User Pool with the following configurations:

Custom password policy (length, complexity requirements)
MFA configuration settings
Account recovery options
Email configuration for user verification



Reviewed and confirmed User Pool creation
Validated the User Pool settings via lab validation
