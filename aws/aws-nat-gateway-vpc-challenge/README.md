NAT Gateway VPC Challenge

Overview

A challenge lab requiring independent configuration of a NAT Gateway within a VPC to enable outbound internet access for EC2 instances in a private subnet, without step-by-step guidance.

Services Used


Amazon VPC
NAT Gateway
Amazon EC2
Internet Gateway
Route Tables


Steps Performed


Created a VPC with public and private subnets
Attached an Internet Gateway to the VPC
Launched an EC2 instance in the private subnet
Created a NAT Gateway in the public subnet with an Elastic IP
Updated the private subnet route table to route internet traffic through the NAT Gateway
Verified outbound internet connectivity from the private EC2 instance


Outcome

Challenge validation passed ✅ — 100%NAT Gateway VPC Challenge
